遊戲營利數據分析與自動化驗證

本專案展示了從 API 數據獲取、匯率轉換、動態報表生成，到 Pytest 自動化測試整合的完整資料工程解決方案。

完成的工作內容

  • 💱

    資料處理與多幣種匯率轉換

    使用 Pandas 抓取 API，並依據特定字典對多種國際幣別進行匯率正規化轉換，確保總盈虧與押注金額在統一基準上計算。

  • 📊

    互動式資料視覺化報表

    使用 Vanilla HTML/CSS/JS 與 Chart.js 構建。包含遊戲與玩家的多維度查詢、各房間類型 (N/PTK/D/K) 的局數比率分析，以及使用幣種的熱度統計。

  • ⚡

    自動化測試與 Allure 整合

    透過 Pytest 對每一款遊戲與每一位玩家進行總營利率閾值驗證 (大於 -2.5% 即觸發異常警告)，並透過 Allure 框架生成專業的測試報告。
